Detailed Description¶

DeprecatedThis profiling utility is deprecated and scheduled for removal in a future release. A more fully featured alternative, likely tracy, will be used instead.

Contains the ScopedProfiler RAII class for timing named code blocks, a simple Timer class for quick timing prints, and the SCOPED_PROFILER macro for convenience.
